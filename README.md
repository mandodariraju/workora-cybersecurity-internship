# 🛡️ Workora Cybersecurity Internship

## 📌 Overview

This repository contains the projects completed during the **Workora Cybersecurity Internship**. The internship focused on developing practical cybersecurity skills through hands-on projects involving password security, network scanning, and phishing email analysis.

Each task demonstrates the implementation of core cybersecurity concepts using Python and follows industry-inspired security practices.

---

## 🎯 Internship Objectives

- Develop secure programming skills using Python.
- Understand password security and entropy calculations.
- Learn network socket programming and port scanning.
- Identify phishing attacks through email analysis.
- Build practical cybersecurity tools for security assessment.

# 📋 Completed Tasks

## 🔐 Task 1 – Password Strength Checker

### Objective

Develop a Python program that evaluates password strength using password policy validation, entropy calculations, and dictionary password detection.

### Features

- Password policy validation
- Password entropy calculation
- Dictionary password detection
- Password strength classification
- Security improvement suggestions

**Technologies Used**

- Python 3
- math
- string

---

## 🌐 Task 2 – Port Status Checker

### Objective

Develop a network socket utility that checks the status of TCP ports on a target host using socket programming.

### Features

- TCP socket connections
- Target IP input
- Custom port range
- Open, Closed, and Filtered port detection
- Scan execution time

**Technologies Used**

- Python 3
- socket
- time

---

## 📧 Task 3 – Email Risk Analyzer

### Objective

Develop a Python tool that analyzes email headers and message content to identify phishing indicators and spoofing attempts.

### Features

- Email header analysis
- SPF validation
- DKIM verification
- Suspicious URL detection
- Phishing keyword detection
- Risk score calculation
- Email security audit report generation

**Technologies Used**

- Python 3
- email
- re

# 🚀 How to Run the Projects

Clone the repository:

```bash
git clone https://github.com/yourusername/workora-cybersecurity-internship.git
```

Navigate to the desired task folder:

```bash
cd Task-1-Password-Strength-Checker
```

or

```bash
cd Task-2-Port-Status-Checker
```

or

```bash
cd Task-3-Email-Risk-Analyzer
```

Run the corresponding Python script:

```bash
python PasswordStrengthChecker.py
```

```bash
python PortStatusChecker.py
```

```bash
python EmailRiskAnalyzer.py
```

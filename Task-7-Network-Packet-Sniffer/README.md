# 🌐 Network Packet Sniffer

## 📌 Overview

The **Network Packet Sniffer** is a Python-based command-line network monitoring tool designed to capture and analyze raw network traffic.

The tool uses raw sockets to inspect network packets, parse IP headers, identify TCP/UDP protocols, extract port information, and generate traffic logs for security analysis.

This project demonstrates basic network monitoring and packet analysis techniques used in cybersecurity.

---

# 🎯 Objective

The objective of this project is to develop a packet inspection utility capable of:

- Capturing raw network packets
- Parsing IP header information
- Identifying network protocols
- Extracting TCP/UDP port details
- Logging network traffic statistics
- Understanding network communication behavior

---

# ✨ Features

- ✅ Raw socket packet capture
- ✅ IP header parsing
- ✅ Source and destination IP extraction
- ✅ TCP protocol analysis
- ✅ UDP protocol analysis
- ✅ Source and destination port detection
- ✅ Network traffic logging
- ✅ Real-time packet monitoring
- ✅ Command-line execution
